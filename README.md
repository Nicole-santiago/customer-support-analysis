
# 📊 Customer Support KPI Analysis

## 📌 Overview
This project analyzes contact center performance using key KPIs such as AHT, CSAT, and Abandon Rate.

## 🛠 Tools Used
- SQL
- Power BI
- Excel

## 📊 Key Insights
- Identified top-performing agents based on CSAT
- Detected trends in average handling time
- Highlighted opportunities to reduce abandonment rate
